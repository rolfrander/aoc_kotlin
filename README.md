# Advent of Code in Kotlin

Learning kotlini and spring boot through Advent of Code

All of this code is hand-written without any agents, but with a lot of help from the kotlin docs, google, stack overflow and ChatGPT.

I have solved many AoC-quests before in my prefered language clojure, but am reimplementing in kotlin to learn a new language.

Using spring boot isn't really well suited for these tasks, command line would be a more natural fit, but exposing solutions as web-pages through spring enables me to get some experience with spring boot as well.
